Sposob uruchomienia zadania 3 z listy 3.

Dla malych danych:

java L3z3 n v

gdzie n to rozmiar tablicy a v to liczba ktorej szukamy.

Przyklad:

java L3z3 20 10


Dla duzych danych:

java L3z3 n v stat

w tym trybie odpali sie program ktory bedzie losowac nastepne v i n jest z podanego zakresu.

Przyklad:

java L3z3 2 1 stat 

w tym przypadku n i v nie maja wiekszego znaczenia bo i tak zostaja zmienione w kazym przejsciu petli.
